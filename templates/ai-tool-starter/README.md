# ai-tool-starter Template

Built on DevArsenal. Ready to customize and ship.

## Quick Start
```bash
cp -r templates/ai-tool-starter apps/my-app
pnpm install
pnpm dev
```

## What's Included
- Pre-configured Next.js 15 app
- Auth, billing, and dashboard wired up
- Domain-specific components and routes
- Ready for Vercel deployment
